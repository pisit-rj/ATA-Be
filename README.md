### Prerequisite

- Java 17
- mvn

### Setup Instructions
1. **Clone the repository:**
 ```bash
   git clone https://github.com/pisit-rj/ATA-Be.git
   cd ATA-Be
````
2. **Build the project:**
 ```bash
   mvn clean install
````
3. **Run the application:**
 ```bash
   mvn spring-boot:run
````
